title: Getting Started
---

Make Starlight sidebar groups clickable by linking them to their index page.

## Features

- Make groups clickable and link to the `index.md` page or page with same name as group (same level)
- If group is autogenerated, automatically remove index page from list and link to it via clickable group instead.
- Customization?: Configure where groups links to? Maybe (for initial release only default to folder link -> `index.md`)

## Prerequisites

You will need to have a Starlight website set up.
If you don't have one yet, you can follow the ["Getting Started"](https://starlight.astro.build/getting-started) guide in the Starlight docs to create one.

## Installation

import { Steps, Tabs, TabItem } from '@astrojs/starlight/components'

<Steps>

1. `starlight-sidebar-clickable-groups` is a Starlight [plugin](https://starlight.astro.build/reference/plugins/). Install it by running the following command in your terminal:

   <Tabs syncKey="pkg">

   <TabItem label="npm">

   ```sh
   npm install starlight-sidebar-clickable-groups
   ```

   </TabItem>

   <TabItem label="pnpm">

   ```sh
   pnpm add starlight-sidebar-clickable-groups
   ```

   </TabItem>

   <TabItem label="Yarn">

   ```sh
   yarn add starlight-sidebar-clickable-groups
   ```

   </TabItem>

   </Tabs>

2. Configure the plugin in your Starlight [configuration](https://starlight.astro.build/reference/configuration/#plugins) in the `astro.config.mjs` file.

   ```diff lang="js"
   // astro.config.mjs
   import starlight from '@astrojs/starlight'
   import { defineConfig } from 'astro/config'
   +import starlightSidebarClickableGroups from 'starlight-sidebar-clickable-groups'

   export default defineConfig({
     integrations: [
       starlight({
   +      plugins: [starlightSidebarClickableGroups()],
         title: 'My Docs',
       }),
     ],
   })
   ```

3. [Start the development server](https://starlight.astro.build/getting-started/#start-the-development-server) to preview the plugin in action.

</Steps>
